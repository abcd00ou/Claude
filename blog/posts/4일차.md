# 삼성 T9 SSD — USB 포트 확인 방법과 Gen 2×2 지원 여부 체크하는 법

삼성 T9을 구매하고 "생각보다 빠르지 않은데?" 하시는 분들이 있습니다.

대부분 원인은 제품이 아닌 PC 포트에 있습니다.
T9의 속도를 제대로 활용하려면 USB 3.2 Gen 2×2 포트가 필요한데,
이걸 확인하지 않고 연결하면 절반 이하 속도만 나올 수 있습니다.

확인 방법을 단계별로 안내해 드리겠습니다.


## ■ USB 규격별 최대 속도 비교

| 규격 | 최대 대역폭 | T9 연결 시 실속도 |
|------|------------|-----------------|
| USB 3.0 / 3.1 Gen 1 | 5Gbps | ~400~500 MB/s |
| USB 3.1 / 3.2 Gen 2 | 10Gbps | ~900~1,000 MB/s |
| USB 3.2 Gen 2×2 | 20Gbps | ~1,800~2,000 MB/s |

T9의 공식 스펙(2,000 MB/s)은 Gen 2×2 연결 기준입니다.


## ■ Windows에서 포트 규격 확인하는 법

**방법 1: 장치 관리자 확인**
1. Win + X → 장치 관리자 열기
2. "범용 직렬 버스 컨트롤러" 항목 펼치기
3. "USB 3.2 Gen 2×2" 또는 "USB xHCI" 문구 확인
4. 해당 항목 더블 클릭 → 속성에서 규격 확인


**방법 2: CPU-Z / HWiNFO 툴 활용**
- HWiNFO → Motherboard 탭 → USB 컨트롤러 항목에서 규격 확인 가능
- 무료 툴이며 설치 없이 포터블 버전으로도 사용 가능


**방법 3: 메인보드 사양서 확인**
- 메인보드 모델명 검색 → 제조사 공식 사이트 스펙 페이지
- "USB 3.2 Gen 2×2" 항목 및 포트 위치 확인


## ■ 어떤 포트에 꽂아야 하나

Gen 2×2는 보통 메인보드 후면 패널의 특정 포트 1~2개에만 지원됩니다.
전면 패널 포트는 대부분 Gen 1~Gen 2 수준입니다.

메인보드 설명서에서 각 포트의 규격을 확인한 후 해당 포트에 연결하세요.
케이블은 T9 동봉 USB-C to C 케이블 사용을 권장합니다.


## ■ 노트북 사용자라면

노트북에서 Gen 2×2를 지원하는 경우는 상대적으로 적습니다.
Thunderbolt 4 포트가 있다면 USB 3.2 Gen 2×2보다 빠른 40Gbps 대역폭을 활용할 수 있어 T9 성능을 충분히 발휘할 수 있습니다.


## ▣ 오늘 내용 요약

1. T9의 최대 속도를 활용하려면 PC에 USB 3.2 Gen 2×2 포트가 있어야 합니다.
2. 장치 관리자 또는 메인보드 사양서에서 포트 규격을 사전 확인할 수 있습니다.
3. 메인보드 후면 패널 특정 포트에만 Gen 2×2가 지원되는 경우가 많으니 위치도 함께 확인하세요.
