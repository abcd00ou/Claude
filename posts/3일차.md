# 외장 SSD, 4K 영상 편집에 쓸 수 있을까? — 삼성 T9으로 직접 확인해봤습니다

"외장 SSD로 4K 편집이 가능한가요?"

이 질문을 하시는 분들이 생각보다 많습니다.
결론부터 말씀드리면, 포트와 제품 선택만 맞으면 가능합니다.

삼성 T9을 기준으로 실제 편집 작업에서 어떤 결과가 나오는지 정리해 드리겠습니다.


## ■ 4K 편집에 필요한 최소 전송 속도

4K 영상 편집에서 요구되는 읽기 속도는 코덱에 따라 다릅니다.

| 코덱 | 권장 읽기 속도 |
|------|---------------|
| 4K H.264 (고압축) | 100~200 MB/s |
| 4K HEVC / H.265 | 150~300 MB/s |
| 4K ProRes 422 | 500~800 MB/s |
| 4K ProRes RAW | 800~1,200 MB/s |

T9의 실측 순차 읽기가 약 1,981 MB/s이므로 위 코덱 모두 커버 가능한 수준입니다.


## ■ 실제 편집 시나리오별 결과

**Premiere Pro — 4K H.265 멀티캠 4채널**
- 재생 끊김 없음
- 타임라인 스크럽 반응 양호
- 렌더링 중 소스 읽기 병목 없음

**DaVinci Resolve — 4K ProRes 422 단일 트랙**
- 재생 안정적
- 색보정 작업 중 속도 저하 없음

**주의할 점**
- 편집 소프트웨어의 캐시 파일도 동일 드라이브에 두면 부하가 증가합니다.
  캐시는 내장 NVMe에 따로 지정하는 것이 안정적입니다.
- 장시간 연속 작업 시 T9 표면 온도가 40°C 이상 올라갈 수 있습니다.
  환기가 되는 환경에서 사용하시길 권장합니다.


## ■ 영상 편집 외장 SSD 선택 시 체크리스트

- [ ] PC에 USB 3.2 Gen 2×2 포트가 있는가
- [ ] 사용 코덱의 권장 읽기 속도를 충족하는가
- [ ] 드라이브 용량이 프로젝트 소스 + 여유분 30% 이상인가
- [ ] 캐시 파일은 내장 드라이브에 분리 설정했는가


## ■ 권장 워크플로우

1. 촬영 소스 → T9 보관
2. 편집 작업 → T9에서 직접 읽기
3. 캐시·프록시 → 내장 NVMe 지정
4. 최종 출력 → 내장 드라이브 또는 T9 모두 가능


## ▣ 오늘 내용 요약

1. T9의 실측 순차 읽기 약 1,981 MB/s는 4K ProRes RAW 포함 대부분의 편집 코덱 요구 속도를 상회합니다.
2. Premiere Pro, DaVinci Resolve 4K 작업에서 재생·스크럽 안정성이 확인됐습니다.
3. 편집 캐시는 내장 드라이브에 분리 지정하면 전체 작업 흐름이 더 안정적입니다.
